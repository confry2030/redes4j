Redes de Computadoras

Origen de las Redes

Las redes de computadoras surgieron en la década de 1960 con el objetivo de compartir información y recursos entre computadoras. Uno de los primeros proyectos fue ARPANET, creado para conectar universidades y centros de investigación. Con el tiempo, estas redes evolucionaron hasta convertirse en el Internet que conocemos hoy.

Definición de Red

Una red de computadoras es un conjunto de dispositivos conectados entre sí con el propósito de compartir información, archivos, recursos y servicios como impresoras o acceso a Internet.

Tipos de Redes

PAN (Personal Area Network)

Red de corto alcance que conecta dispositivos personales como celulares, audífonos o relojes inteligentes.

LAN (Local Area Network)

Red local que conecta computadoras dentro de un área pequeña, como una casa, escuela u oficina.

MAN (Metropolitan Area Network)

Red que cubre una ciudad o zona urbana, conectando varias redes LAN.

WAN (Wide Area Network)

Red de gran alcance que puede cubrir países o continentes, como Internet.

Modelos de Comunicación

Cliente - Servidor

Un dispositivo (cliente) solicita información y otro (servidor) la proporciona. Ejemplo: un navegador web accediendo a una página.

Cliente - Cliente (P2P)

Todos los dispositivos pueden actuar como clientes y servidores al mismo tiempo, compartiendo información directamente.

Topologías de Red

Bus

Todos los dispositivos se conectan a un solo cable principal.

Punto a Punto (P2P)

Conexión directa entre dos dispositivos.

Estrella

Todos los dispositivos se conectan a un dispositivo central como un switch.

Anillo

Los dispositivos se conectan formando un círculo.

Malla

Cada dispositivo está conectado con varios otros, ofreciendo alta seguridad.

Árbol

Combina varias topologías en forma jerárquica.

Mixta

Combinación de diferentes topologías en una sola red.

Medios de Conexión

Medios Guiados (Alámbricos)

  • Cable coaxial
  • UTP (par trenzado)
  • STP (par trenzado blindado)
  • Fibra óptica

Medios No Guiados (Inalámbricos)

  • Infrarrojo
  • Wi-Fi
  • Satélite
  • Microondas
  • Bluetooth

Dispositivos de Interconexión

  • Hub: Envía la información a todos los dispositivos conectados.
  • Switch: Envía la información solo al dispositivo correcto.
  • Router: Conecta diferentes redes entre sí.
  • Bridge: Une dos redes similares.
  • Repetidor: Amplifica la señal para mayor alcance.
  • Transceptor: Envía y recibe señales de red.

Modelo OSI

El modelo OSI es un modelo de referencia que explica cómo se comunican las redes mediante 7 capas:

  1. Física
  2. Enlace de datos
  3. Red
  4. Transporte
  5. Sesión
  6. Presentación
  7. Aplicación
